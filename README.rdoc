== HazTuPregunta.com

== Instalacion

1. Copia el repositorio a tu maquina:
       <tt>Git clone https://github.com/murilloarturo/HazTuPregunta.git</tt> 

2. Cambia al directorio del proyecto:
       <tt>cd HazTuPregunta</tt> 


2. Instala las gemas necesarias:
       <tt>bundle install</tt> 

3. Arranca el servidor de pruebas. (http://localhost:3000/):
       rails s
